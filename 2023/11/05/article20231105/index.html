Abstract的写作要点

不同领域、不同期刊的摘要要求不一样，因此没有固定的格式。

  • 背景+方法+结果（+贡献/意义）

  • 问题+方法+结果（+贡献/意义）

  • 方法+结果（+贡献/意义）

  • …

标题和摘要是文章的核心，但不是全部。

语法注意事项

1. 时态

摘要里面的时态至关重要。当不知道用什么时态的时候，全部用现在时是不会错的。

  • 问题一般用现在时

The main problem, however, is …

We examine why these models have difficulty with …

  • 介绍论文本身的内容一般用现在时

This paper presents a new methodology for …

In this paper we apply …

  • 当提到方法和研究期间做的事情时，用过去时（制备样本）或者现在时（提出理论和分析）

Several models are created using …

The accuracy is evaluated by …

  • 结果一般用现在时，当然也可以使用过去时，这取决于结果的好坏

We show that this theory also applies to …

  • 贡献一般用现在完成时或者现在时

We have demonstrated the feasibility of this approach by…

his process is suitable for the production of…

This framework can be used to evaluate…

This approach can be applied to…

2. 长度

一般来说摘要是150-250个字，需要仔细斟酌。多用定语从句 that/which。
